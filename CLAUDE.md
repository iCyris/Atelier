# Convallaria for Claude Code

Before working in this repository, read `AGENTS.md` at the repository root and follow it as the project authority.

Use the commands in `.claude/commands/` as lightweight Claude Code entry points inside this checkout. User-level installs copy managed command wrappers to `~/.claude/commands/` that point back to this checkout. All commands route to the shared skill at `skills/convallaria/SKILL.md`.

Keep `claude/` as the tracked adapter source. `.claude/` is generated by `python3 scripts/conva.py sync --no-codex-cachebuster` and may contain local settings such as `.claude/settings.local.json`; do not treat `.claude/` as source. Run `python3 scripts/conva.py install` when the commands should also be available from user-level Claude Code command discovery.

Treat `skills/convallaria/subskills/refine/` as the integrated design-system extraction subskill and preserve its workflow carefully when editing it.
